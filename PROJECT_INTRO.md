## 🌟 项目简介

教育参谋系统是一套基于 **OpenClaw 多Agent框架** 构建的智能教育管理平台，专为高中班主任和任课教师设计。系统通过多Agent协同工作，实现学生档案智能管理、风险预警自动监测、教育督导每日复盘等功能，大幅提升教师的工作效率。

### 核心特性

| 特性 | 说明 |
|:-----|:-----|
| 🤖 **多Agent协作** | 10个专业化Agent协同工作，各司其职 |
| 📊 **智能风险预警** | 多维度评估学生风险，自动识别高危学生 |
| 🔍 **每日督导复盘** | 自动生成督导报告，待办事项清晰 |
| 💬 **谈话计划助手** | 智能推荐谈话学生，提供话术建议 |
| 📅 **日历同步** | 与飞书日历同步课表和日程 |
| 📈 **成绩分析** | 自动分析成绩分布，识别学业预警 |
| 🔐 **数据校验** | 多Agent交叉校验，确保数据准确 |
| 💾 **数据安全** | 本地存储，数据完全自主可控 |

### Agent矩阵

| Agent | 角色 | 核心职责 | 定时任务 |
|:------|:-----|:---------|:---------|
| **main** | 主协调 | 消息路由、数据保存 | 实时 |
| **supervisor** | 督导汇总 | 风险评估、报告生成 | 22:00 |
| **validator** | 数据校验 | 数据核验、质量控制 | 每6小时 |
| **academic** | 学业分析 | 成绩分析、学业预警 | 07:05 |
| **psychology** | 心理监测 | 心理危机、风险预警 | 21:00 |
| **safety** | 安全检查 | 实验室安全 | 周一08:00 |
| **home_school** | 家校沟通 | 通知生成、家长联系 | 08:30 |
| **research** | 科研助理 | 论文数据、课题管理 | 22:10 |
| **executor** | 系统维护 | 自维护、错误修复 | 01:00 |
| **talk_planner** | 谈话计划 | 谈话推荐、话术生成 | 按需 |

## 🚀 快速部署

### 一键安装

```bash
# 克隆项目
git clone https://github.com/your-repo/education-advisor.git
cd education-advisor

# 运行安装脚本
bash install.sh

# 按提示配置飞书参数

### 一、系统底层核心能力升级

#### 1.1 全链路数据智能治理能力建设

| 当前痛点 | 升级目标 |
|:---------|:---------|
| 复杂场景下有效数据提取难 | 搭建班级管理专属数据中台 |
| 依赖人工提醒补充数据 | 学生全维度数据自动化分析 |
| 数据分散难以统一调度 | 数据自动定位、精准抓取、清洗、筛选 |

**核心能力**：
- 🎯 学生个体全维度数据标签体系
- 🔗 学生群体关联影响模型
- 📊 数据的自动定位与智能筛选
- 🔄 跨Agent数据实时同步

#### 1.2 多Agent系统实时互通架构升级

```
当前架构          升级后架构
┌─────────┐      ┌─────────┐
│Agent A  │      │ 统一数据总线 │
└────┬────┘      └────┬────┘
     │                 │
┌────▼────┐      ┌────▼────┐
│Agent B  │  →   │Agent A  │
└────┬────┘      ├─────────┤
     │            │Agent B  │
┌────▼────┐      ├─────────┤
│Agent C  │      │Agent C  │
└─────────┘      └─────────┘
  数据互通延迟     实时同步共享
```

**升级目标**：
- ⚡ 统一的实时数据总线
- 🔄 数据跨Agent实时同步
- 🚀 保留精细化处理优势的同时解决延迟问题
- 📈 全面提升系统响应速度与运行稳定性

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐

**核心突破**：
