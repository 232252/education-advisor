## 🌟 项目简介

Education Advisor AI (EAA) 是一套基于 **OpenClaw 多Agent框架** 构建的智能教育管理平台，专为高中班主任和任课教师设计。系统通过多Agent协同工作，实现学生档案智能管理、风险预警自动监测、教育督导每日复盘等功能。

### 核心特性

| 特性 | 说明 |
|:-----|:-----|
| 🤖 多Agent协作 | 10个专业化Agent协同工作，各司其职 |
| 📊 智能风险预警 | 多维度评估学生风险，自动识别高危学生 |
| 🔍 每日督导复盘 | 自动生成督导报告，待办事项清晰 |
| 💬 谈话计划助手 | 智能推荐谈话学生，提供话术建议 |
| 📅 日历同步 | 与飞书/Google等外部日历同步 |
| 📈 成绩分析 | 自动分析成绩分布，识别学业预警 |
| 🔐 数据校验 | 多Agent交叉校验，确保数据准确 |
| 💾 数据安全 | 本地存储，数据完全自主可控 |
| 🗄️ 事件溯源 | Rust CLI（`eaa`），不可变事件流，完全可追溯 |
| 🔌 多通道支持 | 飞书、QQ、Discord、Telegram、微信等 |

## 🚀 快速部署

### 一键安装

```bash
git clone https://github.com/232252/education-advisor.git
cd education-advisor
bash install.sh
```

### 配置

本项目**不绑定任何特定通信平台**。您可以选择飞书、QQ、Discord、Telegram 等任意 OpenClaw 支持的通道。

在 OpenClaw 配置文件中设置您选择的通道即可。参见 [部署指南](./docs/DEPLOYMENT.md)。
