CORE FEATURES

六件事，我们做得不一样

Education Advisor 不是把"通用聊天机器人"套上教育壳子，而是从数据层、隐私层、调度层就把教育场景当一等公民来设计。

事件溯源 + 不可篡改
每一个操行事件都是 append-only 的事实，可以回滚但不能删除。家长质问"为什么扣分"，整条审计链一键导出。
AES-256-GCM 隐私脱敏
学生姓名、身份证、电话、地址在落盘前全部加密；进 LLM 前自动替换为 S_017、ID_42、1XX-XXXX-1234。
18 个 AI 智能代理
班主任助手、风险预警、纪律员、家校沟通员、日报周报生成…每个 agent 各司其职，最小权限隔离。
30+ LLM 全兼容
OpenAI、Anthropic Claude、Google Gemini、DeepSeek、通义千问、豆包、智谱 GLM、Kimi…任何 OpenAI 兼容端点都能挂。
Rust 数据引擎
19.5 MB 的 Rust 二进制（6 平台支持），原子写、文件锁、强类型校验。桌面端只是"视图 + 编排器"，写入永远经过它。
本地优先 + 飞书同步
全功能离线运行；飞书 Bitable 同步是可选的，CLI 钩子 + 定时任务双保险，永远不会循环同步。
DIFFERENTIATORS

五个差异点，决定了能不能真的用起来

不是 PPT 上的功能列表，是从两条真实班级的日常使用里抽出来的关键决策。

01

3–4B 小模型也能跑

所有 agent 的 prompt 都按"Qwen 3.5 4B 在 6 GB 显卡上也能用"为前提设计。规则是：所有数字必须来自工具调用、所有写入必须经过授权校验、所有输出必须符合 JSON schema。Qwen 4B 完全离线，整套系统可跑。

# 强类型校验规则
rule: "所有数字必须可回溯到工具调用"
rule: "所有写入操作必须经过 eaa-cli"
rule: "PII 字段必须先脱敏再送 LLM"
tier: low-cost | high-quality
max_context: 8192 tokens
02

PII 引擎 + 审计日志

AES-256-GCM 加密映射表，Argon2 派生主密码。每一次脱敏/还原都记入审计。家长报告出库时你可以"定向还原"指定学生的姓名；不勾选的学生永远是 S_017。

$ eaa privacy anonymize "Alice 的身份证是 110101200501012345"
→ "S_017 的身份证是 ID_042"   # 已脱敏

$ eaa privacy deanonymize --student S_017 --for parent
→ "Alice 的身份证是 110101200501012345"
03

一条命令装好

Windows 双击安装包就完事；macOS / Linux 也是一行命令。Rust 二进制和 eaa-cli 自动捆绑，零外部依赖。日志、缓存、SQLite 数据库都走标准 userData 目录，升级不会丢数据。

# Windows
.\Education.Advisor-0.1.0-rc.1-Setup.exe

# macOS (Homebrew)
brew install education-advisor

# Linux (AppImage)
chmod +x Education.Advisor-0.1.0-rc.1.AppImage
./Education.Advisor-0.1.0-rc.1.AppImage
04

18 个 agent 协同

class-monitor 写入事件 → risk-alert 巡检 → counselor 生成谈话草稿 → feishu-sync 推到家长端。一条数据，五种角色，没有一个能干超出权限的事。

       ↓
risk-alert     → 14 日滚动巡检
       ↓
counselor      → 生成 3 份谈话草稿
       ↓
feishu-sync    → 推到家长 Bitable

FAQ

常见问题

这个项目是"基于"某个上游框架吗？

不是。整个项目（桌面端、18 个 agent、Rust 数据引擎、隐私脱敏引擎、Electron 壳、React 渲染层）都是这个仓库自研的代码，MIT 协议开源。不依赖任何上游框架。

学生数据会上云吗？

默认不会。所有学生姓名、ID、电话、地址都用 AES-256-GCM 加密后存在本机。送到 LLM 的文本会先脱敏（Alice → S_017）。家长消息出库时由你手动决定"定向还原"。审计日志全程记录。

小模型（4B）真的能跑得动吗？

能。每个 agent 的 prompt 都按"3–4B 模型也能跑"为前提设计。规则是：所有数字必须来自工具调用，所有写入必须经过授权校验，所有输出必须符合 JSON schema。Qwen 4B 跑在 6GB 显卡上，整套系统可以完全离线运行。

支持哪些 LLM？

通过 @earendil-works/pi-ai SDK 统一接入 30+ 提供商：OpenAI、Anthropic、Google Gemini、Mistral、DeepSeek、Qwen（DashScope）、Doubao（Volcengine）、Zhipu GLM、Moonshot Kimi，以及任何 OpenAI 兼容端点（Ollama、LM Studio、vLLM、llama.cpp）。

可以删除 Rust 引擎用别的数据存储吗？

可以。桌面端对 Rust 引擎的调用集中在 src/main/services/eaa-bridge.ts 一个文件。换成 PostgreSQL / Firestore / 任何其他存储引擎只需重写这一个文件。Agent prompt 和 UI 不动。

怎么自己加一个新的 agent？

在 agents/your-id/ 下放两个 Markdown 文件（SOUL.md + AGENTS.md），再到 config/agents.yaml 加一条注册项，重启应用。就这样。

商业使用可以吗？

可以。项目以 MIT 协议发布，可商用、修改、闭源分发，仅需保留版权声明即可。详见 LICENSE。
