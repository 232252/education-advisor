# 安全限制

本系统为教育管理辅助工具，所有使用者和部署者必须遵守以下安全限制：

## 数据访问限制
1. CLI是唯一数据读写入口，禁止直接修改JSON文件
2. 新增事件必须通过 `copaw add` 命令，由Rust强制校验
3. 禁止绕过校验机制修改数据

## 系统安全限制
1. 本系统不得用于操控宿主系统
2. 禁止通过本系统执行系统命令
3. 禁止访问非教育数据相关的系统资源
4. AI Agent仅限读写教育数据，不得越权

## 数据安全
1. 学生数据属于敏感信息，部署者需遵守相关法律法规
2. 禁止将真实学生数据上传至公开仓库
3. 示例数据仅供测试使用

## Agent行为限制
1. Agent只能通过CLI读写数据
2. Agent不得自主修改系统配置
3. Agent不得执行未经授权的操作
4. 所有Agent行为必须有日志记录
